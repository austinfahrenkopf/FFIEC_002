## Caveats for a truly fresh rebuild

1. **Panel parquet** (27.7 MB) is provided in this kit. A fresh rebuild requires running
   steps 1–4 in RUNBOOK.md. Step 2 (`build_ffiec002_overnight.py`) takes ~1.5 hours and
   requires Playwright + real Chrome for the NIC phase (2021 Q3+).

2. **Hierarchy** is reproducible from the PDF and overrides. Running `build_hierarchy_002.py`
   will regenerate `ffiec002_hierarchy.json` deterministically given the same
   `FFIEC002_202606_f.pdf` and `ffiec002_hierarchy_overrides.json`.

3. **`_qa_final.py`** is designed to run from the `External Bank Data\` workspace root,
   not from reproduce/. Its hardcoded paths reference `FFIEC 002\site_002\index.html`
   relative to that root. Running it from a standalone clean-room requires adjusting
   those paths.

4. **Chicago Fed source.** Steps 1–2 use the Chicago Fed public file server (not
   Akamai-guarded). `download_ffiec002_playwright.py` uses Playwright for NIC data
   (2021 Q3+). Do not use archive.org mirrors — use the official Chicago Fed URLs only.
